## Getting Started

To run the application locally, follow these steps:

1. **Clone the repository:**
  
   ```bash
   git clone https://github.com/itsmdshahin/todo.git
   cd todo
   npm install

2 . Start the server:
   ```bash
   npm run dev
   ```
